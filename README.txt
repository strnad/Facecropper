FaceCropper

python crop_and_resize.py <input_path> <output_path> --offset_x <offset_x> --offset_y <offset_y> --face_percent <face_percent> --resize <resize>

This will process the images in the input_images folder, save the cropped and resized images in the output_images folder, apply a 10% horizontal offset and a 20% vertical offset around the face, make the face size 80% of the cropped photo, and resize the output image to 256x256 pixels.
